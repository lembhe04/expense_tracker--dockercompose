# 💰 Expense Tracker

A simple full-stack Expense Tracker built using Next.js, PostgreSQL, and Docker.

---

## 🚀 Features

- User Signup
- User Login
- Add Expense
- View Expenses
- Update Expense
- Delete Expense
- Monthly Total Calculation

---

## 🛠️ Tech Stack

- Next.js (App Router)
- PostgreSQL
- Docker
- Tailwind CSS

## 🐳 Run With Docker

Start the application:
docker-compose up --build

To Run in background:

docker-compose up -d
